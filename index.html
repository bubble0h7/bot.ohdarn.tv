OH DARN BOT

An open source general purpose Discord bot

FEATURES

Default Auto-Role

Set your own custom default server role and let Oh Darn Bot handle assigning it to new members.

Currently Streaming

Automatically assigns users a role when they go live, and removes it when they stop. Great for supporting and promoting your server's streamers.

Random Generators

Emulate rolling dice or let fate decide your next champion to play in League of Legends.

Moderating Tools

Designed to make moderating easier, like clearing large quantites of messages in one go.

GETTING STARTED

Invite

  • Invite Oh Darn Bot to your Discord Server.
  • You'll need admin permissions in a server to invite the bot.
  • Oh Darn Bot will create it's own text channel for bot commands. Feel free to delete this channel and use your own.

Update the bot's role

  • Make sure the bot's role is at the top of your server's role hierarchy. This is important for the functionality of the bot.
  • At the very least, the bot needs to be above any roles that you wish for it to be able to have an effect on.

Configure

  • Run through the initial configuration. To get started, just use the configuration command below.
